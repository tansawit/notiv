# Support

## Getting Help

For usage questions, bug reports, and feature requests:

- Open a GitHub issue in this repository with clear reproduction steps.
- Include browser version, OS, and extension version (`manifest.json` version).
- Include relevant screenshots or logs when possible.

## Security Issues

Do not open public issues for vulnerabilities.
Follow the reporting process in `SECURITY.md`.

## Before Opening An Issue

- Confirm you are on the latest mainline.
- Run `pnpm run check` and `pnpm run build`.
- Verify OAuth setup (bundled client ID or Settings override) and redirect URI configuration.
