# Code Of Conduct

This project adopts the Contributor Covenant Code of Conduct v2.1.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and constructive in discussions and reviews.
- Accepting and giving feedback professionally.
- Prioritizing user safety, privacy, and security in technical decisions.
- Assuming good intent while still holding high quality standards.

Examples of unacceptable behavior include:

- Harassment, intimidation, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without explicit permission.
- Deliberate disruption of collaboration workflows.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards.
They may remove, edit, or reject comments, commits, issues, or other contributions
that violate this Code of Conduct.

## Reporting

Please report conduct issues privately to the maintainers.
Security issues should follow `SECURITY.md`.
